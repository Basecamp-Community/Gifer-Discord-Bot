## Getting Started with Project
1. Clone the Repo
```bash
git clone https://github.com/Basecamp-Community/Gifer-Discord-Bot.git

cd Gifer-Discord-Bot
```
2. Install Dependencies
```bash
npm install
```
3. Start the Local Server
```bash
node src/bot.js
```
